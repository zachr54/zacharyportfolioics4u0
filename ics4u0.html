Parallel Arrays Example

In this group assignment with Usayd I and Pranav V, we stored computer data using parallel arrays.

Setting up the parallel arrays

ComputerPart1.java 
String[] name = {"HP", "Acer", "Apple", "Lenovo", "Asus"};
String[] GPU = {"RTX 3050", "RTX 5080", "RTX 3060", "RTX 4060", "RX 9060"};
String[] CPU = {"i7", "i5", "i9", "Ryzen 3", "Ryzen 5"};
int[] RAM = {16, 32, 8, 16, 16};
int[] batteryLife = {6, 7, 10, 12, 5};
int[] yearMade = {2020, 2016, 2024, 2022, 2025};

Displaying all data with a for loop

ComputerPart1.java 
public void displayAllData() {
    for (int i = 0; i < name.length; i++) {
        System.out.println("Computer - " + name[i]);
        System.out.println("CPU - " + CPU[i]);
        System.out.println("GPU - " + GPU[i]);
        System.out.println("RAM - " + RAM[i] + "GB");
        System.out.println("Battery Life - " + batteryLife[i] + " hours");
        System.out.println("Year Made - " + yearMade[i]);
        System.out.println();
    }
}

Finding the best computer by comparing a score

ComputerPart1.java 
public void displayTheBest() {
    int bestComp = 0;
    int bestScore = 0;

    for (int i = 0; i < name.length; i++) {
        int score = RAM[i] + batteryLife[i] + yearMade[i];

        if (score > bestScore) {
            bestScore = score;
            bestComp = i;
        }
    }

    System.out.println("The best computer is " + name[bestComp]);
}

Fantastical Invention — Object-Oriented Programming

For this assignment, I built an ElementalBlade class to represent a fantasy sword. This taught me how to group related data and behaviour into one class, use private fields for encapsulation, and control access through getters and setters.

Private fields and constructor

ElementalBlade.java 
public class ElementalBlade {
    private final String swordBaseMaterial;
    private final Color colour;
    private final boolean loyalty;
    private final double lengthCM;
    private final double weightKG;

    private int durability;
    private String swordName;
    private boolean dualWield;
    private int bladeSharpness;
    private int manaLevel;

    public ElementalBlade(String sBM, Color colo, boolean loy,
            double lngCM, double wCM, String swordN, boolean dW) {
        swordBaseMaterial = sBM;
        colour = colo;
        loyalty = loy;
        lengthCM = lngCM;
        weightKG = wCM;
        durability = 100;
        swordName = swordN;
        dualWield = dW;
        bladeSharpness = 100;
        manaLevel = 100;
    }
}

Getters, setters, and validation

ElementalBlade.java 
public int getDurability() {
    return durability;
}

public void setDurability(int dur) {
    if (dur > 100) {
        durability = 100;
    } else if (dur < 0) {
        durability = 0;
    } else {
        durability = dur;
    }
}

public void setBladeSharpness(int bladeS) {
    if (bladeS > 100) {
        bladeSharpness = 100;
    } else if (bladeS < 0) {
        bladeSharpness = 0;
    } else {
        bladeSharpness = bladeS;
    }
}

Methods that change object state

ElementalBlade.java 
public void attack() {
    if (manaLevel >= 5) {
        durability -= 5;
        bladeSharpness -= 1;
        System.out.println("The enemy has been hit.");
    } else {
        System.out.println("Not enough mana!");
    }
}

public void toggleWield() {
    dualWield = !dualWield;
    if (!dualWield) {
        System.out.println("The Sword is now Single Wield.");
    } else {
        System.out.println("The Sword is now Dual Wield");
    }
}

Inferno Blade — Inheritance

Building on ElementalBlade, I created InfernoBlade as a subclass. Inheritance let me reuse the base sword's fields and methods while adding fire-specific properties and abilities. The subclass calls super() in its constructor to initialize the parent class first.

Extending the parent class

InfernoBlade.java 
public class InfernoBlade extends ElementalBlade {
    private final String gemstone;
    private final String element;
    private final int flameSpawnRate;
    private final int glowIntensityLux;
    private final int burnRate;

    private int heatIntensity;
    private int projectileSpeed;
    private String[] flameTypes = new String[5];
    private int fireMasteryLevel;
    private int instabilityLevel;

    public InfernoBlade(String sBM, Color colo, boolean loy,
            double lngCM, double wCM, String swordN, boolean dW,
            String gemS, String el, int fSR, int gIL, int bR,
            int hI, int pS, int fML, int instL) {
        super(sBM, colo, loy, lngCM, wCM, swordN, dW);
        gemstone = gemS;
        element = el;
        flameSpawnRate = fSR;
        glowIntensityLux = gIL;
        burnRate = bR;
        heatIntensity = hI;
        projectileSpeed = pS;
        fireMasteryLevel = fML;
        instabilityLevel = instL;
    }
}

Subclass-only methods using inherited fields

InfernoBlade.java 
public void flameBurst() {
    if (getManaLevel() >= 25) {
        setManaLevel(getManaLevel() - 25);
        heatIntensity += 20;
        instabilityLevel += 10;
        System.out.println("A massive flame burst erupts from the blade!");
    } else {
        System.out.println("Not enough mana!");
    }
}

public void overheatBlade() {
    if (getManaLevel() >= 40) {
        setManaLevel(getManaLevel() - 40);
        heatIntensity += 30;
        instabilityLevel += 20;
        System.out.println("The blade begins to overheat!");
        if (instabilityLevel >= 90) {
            System.out.println("The blade has become highly unstable!");
            setDurability(getDurability() - 50);
        }
    } else {
        System.out.println("Not enough mana!");
    }
}

Using both parent and child behaviour together

InfernoBlade.java 
public void flingBlade() {
    if (getManaLevel() >= 30) {
        setManaLevel(getManaLevel() - 30);
        heatIntensity += 30;
        instabilityLevel += 10;
        if (getDualWield()) {
            System.out.println("The blade was flung, spit "
                + flameTypes[(int) (Math.random() * 5)]
                + " at the enemy and returned!");
        } else {
            System.out.println("The blade was flung, spit "
                + flameTypes[(int) (Math.random() * 5)]
                + " at the enemy and returned!");
        }
        if (instabilityLevel > 90) {
            System.out.println("The blade has become highly unstable!");
        }
    }
}
